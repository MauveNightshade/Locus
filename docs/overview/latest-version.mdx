## 变更列表

### 新增

- Unity 插件安装包改为内置合并后的 Roslyn 运行组件，减少项目内依赖冲突。

### 修复

- 修复部分 Unity 项目安装插件后出现 Roslyn 相关编译冲突的问题。
- 修复应用更新后 Unity 插件资源可能沿用旧文件的问题。
- 修复 Unity 嵌入窗口在父窗口隐藏、切换或恢复时仍可能显示的问题。
- 修复 Unity Editor 快速发送窗口同步消息时偶发连接到旧控制通道的问题。
- Diff 预览会忽略仅由 Windows/macOS 行尾差异造成的改动，减少无意义的变更噪音。

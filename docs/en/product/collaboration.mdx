Locus builds a relatively complete collaboration interface around Git, so writing changes, reviewing diffs, resolving conflicts, and submitting results can all stay in the same workspace.

## Current Collaboration Capabilities

- Repository detection and initialization
- Unity project `.gitignore` initialization
- History graph and branch view
- staged / unstaged / conflict workspaces
- Commit details, stash, and submodule panels
- Built-in Git Terminal
- AI-assisted commit messages
- File diff and Unity semantic diff integration

## Why This Collaboration Style Matters

For Unity teams, the hardest changes to review are often not pure code, but mixed modifications across assets and scripts. Locus puts Git workflows and Unity semantic views in the same interface, which reduces context switching and review cost.

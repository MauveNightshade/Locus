## Changes

### Added

- The Unity plugin package now includes a merged Roslyn runtime component to reduce dependency conflicts inside Unity projects.

### Fixed

- Fixed Roslyn-related compile conflicts that could appear after installing the Unity plugin in some projects.
- Fixed app updates leaving older bundled Unity plugin files in place.
- Fixed cases where the Unity embedded window could remain visible after its parent window was hidden, switched, or restored.
- Fixed an occasional Unity Editor control-channel mismatch when window sync messages were sent quickly.
- Diff previews now ignore changes caused only by Windows/macOS line endings, reducing noisy file changes.

# Frontend Development Guidelines

> Vue 3 + TypeScript frontend conventions for `src/`.

---

## Overview

The frontend is a Vue 3 application using Vite 6, Pinia 3, TypeScript 5.6, and custom UI components. ~40 top-level components, ~10 Pinia stores, ~55 IPC services, and ~60 composables.
